# CodeFlow AI - Landing Page

A responsive SaaS landing page built with React and Tailwind CSS.

## Tech Stack
- React
- Tailwind CSS
- Vite
- Lucide React
- React Syntax Highlighter

## Sections
- Navbar with scroll effect
- Hero section
- Features section
- Pricing plans
- Testimonials
- Footer

## Getting Started
```bash
npm install
npm run dev
```
